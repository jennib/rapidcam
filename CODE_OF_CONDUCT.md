# Code of Conduct

## Our commitment

This project is a small, focused tool. Interactions here — in issues, pull requests, and discussions — should be straightforward and respectful. Treat other contributors the way you would want to be treated in a professional setting.

## What we expect

- Clear, specific bug reports and feature requests.
- Constructive feedback on pull requests.
- Respectful disagreement — it's fine to have different opinions about design decisions.

## What we won't tolerate

- Personal attacks, harassment, or discriminatory language.
- Spam or off-topic noise.
- Publishing anyone's private information without consent.

## Enforcement

Violations can be reported to **jenni.b@gmail.com**. Persistent or severe violations will result in a permanent ban from the repository.
