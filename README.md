# RapidCAM

RapidCAM is a web-based parametric 2D vector editor designed as the foundation for a CNC CAM program. 

**Live Demo:** [https://rapidcam.mycnc.app](https://rapidcam.mycnc.app)

## Features

- Parametric 2D vector editing directly in the browser.
- Typography support, including standard OpenType parsing and single-line (Hershey) fonts for plotting and engraving.
- Robust polygon operations (clipping, offsetting) using Clipper2.
- Fully typed architecture built with TypeScript and Vite.

## Setup

Ensure you have [Node.js](https://nodejs.org/) installed, then install the project dependencies:

```bash
npm install
```

## Development Commands



Run the TypeScript type checker:

```bash
npm run typecheck
```

Build the application for production:

```bash
npm run build
```

Start the local development server:

```bash
npm run dev
